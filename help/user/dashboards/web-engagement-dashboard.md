# Instrumentpanel för webbengagemang

Instrumentpanelen för webbengagemang ger insyn i hur besökarna interagerar med nyckelinnehållet. Det segmenterar data mellan kontobranscher och regioner för att hjälpa er att förstå engagemangstrender. Använd den här instrumentpanelen för att stödja strategiskt beslutsfattande genom att surfa på webbbeteendemönster som bygger på innehållsstrategi och målinriktning av konton.

Om du vill komma åt _webbengagemangsinstrumentpanelen_ markerar du **[!UICONTROL Dashboard]**-objektet i den vänstra navigeringen. Välj sedan fliken **[!UICONTROL Web engagement]** överst på sidan.

![Kontrollpanel för webbengagemang](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## Filtrera data

Klicka på ikonen _Filter_ ( ![Filterikon](../assets/do-not-localize/icon-filter.svg) ) längst upp till vänster om du vill filtrera de data som visas med något av följande attribut:

* **[!UICONTROL Account Region]** - Filtrerar data efter en eller flera valda geografiska regioner som är associerade med konton.
* **[!UICONTROL Account Industry]** - Filtrerar data efter en eller flera valda branschklassificeringar som är associerade med konton.
* **[!UICONTROL Date Range]** - Filtrerar data efter det valda datumintervallet. Standardintervallet är aktuellt datum.

![Filtrera visade data efter attribut](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

Välj så många värden för varje attribut som du vill använda för att filtrera data och klicka på **[!UICONTROL Apply]**.

## [!UICONTROL Top page views] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="Övre sidvyer"
>abstract="De mest visade sidorna på webbplatsen, vilket hjälper dig att identifiera vilket innehåll som genererar mest trafik."

I den här tabellen visas de 10 mest visade webbsidorna, vilket hjälper dig att identifiera vilket innehåll som passar bäst för besökarna. Informationen innehåller följande:

| Kolumn | Beskrivning |
| ------ | ----------- |
| Sidnamn | Webbsidans namn eller titel. |
| Totalt antal visningar | Det totala antalet gånger som sidan visades. |
| Kända besökare (%) | Procentandel av sidvisningar som tilldelas kända (identifierade) besökare. |
| Okända besökare (%) | Procentandel av sidvisningar som tilldelas okända (anonyma) besökare. |

![Tabell för översta sidvisningar](./assets/web-engagement-dashboard-top-page-views.png){width="650" zoomable="yes"}

## [!UICONTROL Page views by account region] {#page-views-by-region}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_page_views_by_region"
>title="Sidvyer per kontoregion"
>abstract="Webbbesöksdistribution segmenterad efter den geografiska regionen för associerade konton."

Den här visualiseringen visar antalet besökare segmenterade efter kontoregionen. Det visar hur webbtrafiken varierar mellan olika geografiska regioner, vilket gör att ni kan skräddarsy innehåll och kampanjer för regionala målgrupper. Håll pekaren över ett streck i diagrammet för att se detaljer, inklusive:

* Namn på kontoregion
* Antal sidvyer

![Sidvisningar efter kontoregion](./assets/web-engagement-dashboard-page-views-by-account-region.png){width="500" zoomable="yes"}

## Engagera med data

Använd menyn _Mer_ (**..**) längst upp till höger i varje diagram och välj **[!UICONTROL View more]** om du vill visa utökade data och insikter.

Popup-fönstret som visas innehåller ett diagram och en tabell som visar hur data har delats upp.

Om du vill hämta data klickar du på **[!UICONTROL Download CSV]** längst upp till höger i datatabellen.

![Övre sidvy, tabell Visa mer-popup - Klicka på Hämta CSV](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}
